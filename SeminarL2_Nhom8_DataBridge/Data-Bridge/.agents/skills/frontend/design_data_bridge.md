# Data Bridge Design System

## 1. Theme & Colors
- **Mode:** Light Mode (SaaS Minimalist).
- **Background:** Video background with a balanced light overlay (`rgba(255, 255, 255, 0.15)` to `rgba(248, 250, 252, 0.7)`).
- **Text Primary:** Slate Dark (`#0f172a`).
- **Text Secondary:** Slate Gray (`#475569`).
- **Text Muted:** Light Slate (`#94a3b8`).
- **Primary Accent (Buttons/Links):** Vibrant Blue (`#2563eb` to `#4f46e5` gradient).
- **Borders:** Very light gray (`#e2e8f0`).

## 2. Typography
- **Font Families:** Plus Jakarta Sans or Inter (sans-serif).
- **Headings:** Bold, tight tracking (`letter-spacing: -2px`), dark slate color.
- **Body:** Clean, legible, 14px to 16px size.

## 3. UI Components & Layout
- **Navigation Bar (Floating Pill):**
  - Position: Fixed at top, floating pill shape.
  - Background: White with opacity (`rgba(255, 255, 255, 0.85)`) and `backdrop-filter: blur(12px)`.
  - Border-radius: `9999px`.
  - Shadow: Soft shadow.
- **Hero Section:**
  - Layout: Centered.
  - Badges: Small rounded pill badge (blue background `#eff6ff`, blue text `#1e40af`).
  - Search Input: Large white card, rounded `24px`, shadow `0 20px 40px -12px rgba(0, 0, 0, 0.08)`.

## 4. Branding
- **Name:** Data Bridge
- **Logo:** Sparkle SVG icon + "Data Bridge" text.
