## 🎯 Tổng Quan Dự Án & Định Hướng Nghiên Cứu

Repository này được xây dựng và phát triển xuyên suốt kỳ học Seminar Chuyên ngành dưới sự hướng dẫn khoa học của **TS. Đỗ Như Tài**. Nội dung chính xoay quanh 2 định hướng công nghệ mũi nhọn:

1. **Generative AI & Agentic Workflows**: Ứng dụng GitHub Copilot, tối ưu hóa Prompt Engineering nâng cao, xây dựng luồng tự động hóa tương tác qua n8n và tích hợp Multi-Agent Systems xử lý thông tin thông minh.
2. **Kiến Trúc Microservices**: Phát triển hệ thống phân tán thực tiễn thông qua dự án **DataBridge (Agent SQL - Multi-Agent NL2SQL System)**. Giao tiếp qua API Gateway, cơ chế Hot-reload đồng bộ, kết nối bảo mật tới các nguồn dữ liệu đa dạng (PostgreSQL, MySQL, SQLite, MongoDB, Redis).

## 🏗️ Hai Cột Mốc Đồ Án Lớn (DataBridge - Agent SQL)

Bên cạnh tiến trình theo tuần, tác giả đã phát triển và tích hợp hai giai đoạn hoàn thiện của hệ thống **DataBridge - Agent SQL**:

### 🔹 [Giai Đoạn 1 - SeminarL1_Nhom8_DataBridge](./SeminarL1_Nhom8_DataBridge/)
- Khởi tạo kiến trúc nền tảng, thiết kế cơ sở dữ liệu mẫu.
- Phác thảo đặc tả hệ thống và sơ đồ luồng dữ liệu (Dataflow Diagram).

### 🔹 [Giai Đoạn 2 - SeminarL2_Nhom8_DataBridge](./SeminarL2_Nhom8_DataBridge/)
Hệ thống **Multi-Agent NL2SQL** hoàn chỉnh cho phép chuyển đổi ngôn ngữ tự nhiên thành câu lệnh truy vấn SQL chính xác và thực thi trực tiếp trên Supabase.
- **API Gateway (Port 8000)**: Điều phối dịch vụ và ghi nhận lịch sử truy vấn.
- **NL2SQL Service (Port 8002)**: AI Engine phân tích ngữ cảnh schema và sinh SQL tự động.
- **Query Service (Port 8001)**: Kết nối bảo mật và thực thi truy vấn PostgreSQL.
- **Frontend Dashboard (Port 3000)**: Giao diện Next.js trực quan hiển thị biểu đồ và bảng kết quả.
