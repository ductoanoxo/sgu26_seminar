==========================================
CHỤP HỲ: TIPS & CONVENTIONS
==========================================

📸 Format & Naming:
   - Format: PNG
   - Resolution: 1920x1080 (standard screen)
   - Naming: lib15_## OR lab15_NNN_description.png (lowercase)
   - Example: lab151_01_code_overview.png

🎨 Code Display:
   - Font size: 14-16pt (readable)
   - Theme: Dark (recommended) hoặc Light
   - Show line numbers: ✅ Yes
   - Word wrap: ✅ Yes cho readability

📝 Terminal:
   - Clear before screenshot: clear (Linux) hoặc cls (Windows)
   - Show full path: export PS1="\w$ " (bash)
   - Font: Monospace 14pt
   - Background: Dark (contrast)

🔑 Important Output:
   - Always capture timestamps
   - Include error messages (if applicable)
   - Show statistics/metrics at end
   - Scroll to show important parts

⚠️ Security:
   - BLUR or HIDE: API keys, passwords
   - Use placeholder: ...
   - Show first 10 + last 10 characters only
   - Example: AIzaSy...GpmR7O

📐 Composition:
   - Include full file path/name in screenshot
   - Show window title bar
   - Include line numbers for code
   - Include timestamps for terminal

✏️ Annotations (Optional):
   - Draw arrows to important code
   - Highlight error messages
   - Circle critical output
   - Add brief labels

📋 Quality Checklist:
   - ✅ Image is sharp and readable
   - ✅ Text is not cut off
   - ✅ Relevant parts are visible
   - ✅ No blur or compression artifacts
   - ✅ Consistent filename naming
   - ✅ Organized in image/ folder

TOOLS TO HELP WITH SCREENSHOTS
==========================================

Windows 10/11:
   - Win+Shift+S: Snipping Tool
   - Win+G: Xbox Game Bar
   - VS Code: Screenshot extension

PowerShell to organize:
   - Get-ChildItem *.png | Measure-Object

Python to batch process:
